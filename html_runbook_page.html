HTML, CSS & JavaScript — The Engine Behind Infrastructure Initiative

Infrastructure Initiative isn’t built with a framework or a game engine — it’s engineered from scratch using pure HTML, CSS, and JavaScript working together as a full front‑end stack.

HTML — The Structural Backbone

HTML provides the architecture of the entire game. It defines the <canvas> battlefield, the UI overlays, the score panel, and the event hooks that JavaScript listens to.

  • The canvas is the rendering surface for all gameplay.
  • UI elements (score, health, warnings) are HTML nodes styled with CSS.
  • Keyboard and mouse events originate from the HTML document.

CSS — The Cinematic Layer

CSS transforms the raw HTML into a neon‑lit, high‑contrast interface. It controls glow effects, transitions, damage flashes, and the dark‑themed control panel.

JavaScript — The Game Engine

JavaScript drives the game loop, physics, collisions, enemy spawning, and difficulty scaling.

How Shots Are Created

When the player fires, JavaScript creates a “shot object” with:

  • starting coordinates
  • velocity
  • direction

Each frame, the shot moves upward and checks for collisions with enemy hitboxes.

Enemy Triangles — Built From Geometry

Enemies are drawn as triangles using canvas path commands:

  • Top point: (x, y)
  • Bottom-left: (x - 15, y + 25)
  • Bottom-right: (x + 15, y + 25)

JavaScript redraws them every frame, creating smooth downward motion.

HTML Functionality — The Hidden Power

While JavaScript handles movement and collisions, HTML is responsible for the game’s structure, accessibility, and responsiveness. It provides the semantic foundation that makes the entire experience stable and scalable.

  • Canvas Element: The entire game world is rendered inside a single HTML tag.
  • UI Panels: Score, health, and warnings are HTML elements layered above the canvas.
  • Event Handling: HTML captures keyboard and mouse events before JavaScript processes them.
  • Responsive Layout: HTML ensures the game scales properly on different screen sizes.
  • Document Flow: The structure allows CSS and JS to interact cleanly and predictably.

Why This Matters

Infrastructure Initiative proves that HTML, CSS, and JavaScript — when used together — can create a fully interactive, animated, responsive game without any external libraries or engines.
